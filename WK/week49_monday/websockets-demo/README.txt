steg 1) öppna mappen websockets-demo i VSCode

------------------------------------------------------------------

steg 2) starta server - CTRL+SHIFT+Ö eller högerklicka på server.js och välj "Open in integrated terminal" - skriv:

node server
nodemon server

nodemon behöver vara installerat globalt, kör i terminalen:

npm i -g nodemon

------------------------------------------------------------------

steg 3) starta klienten - ALT+L, ALT+O eller högerklicka på index.html och välj "Open with live server"

Live Server behöver installeras som extension, länk här: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
