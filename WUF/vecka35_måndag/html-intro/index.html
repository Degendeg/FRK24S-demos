HTML (HyperText Markup Language) är standardmärkspråket som används för att skapa webbsidor. Det består av element som representerar olika typer av innehåll, som text, bilder och länkar. Dessa element strukturerar och formaterar innehållet för visning i webbläsare.

Grundläggande HTML-element:

  • <h1> - <h6>: Rubriker, från nivå 1 (störst) till nivå 6 (minst).
  • <p>: Paragrafer för att gruppera text.
  • <a>: Hyperlänkar för att länka till andra sidor eller resurser.
  • <img>: Bild-element för att inkludera bilder.
  • <ul> & <li>: Oordnad lista med listpunkter.
